## Build Setup

This is how you get things running:

### Web-App
```bash
$ cd whoneeds/app

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```

### API
```bash
$ cd whoneeds-api

#install dependencies, build and run the service at localhost:9000
$ ./gradlew bootRun
```
